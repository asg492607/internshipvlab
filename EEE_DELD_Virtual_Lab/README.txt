EEE + DELD Virtual Laboratory - MIT ADT University

How to open:
1. Extract this ZIP.
2. Open index.html in Chrome, or open the folder in VS Code and run Live Server.
3. Use Subjects to open EEE or DELD experiments.
4. Each experiment contains Aim, Theory, Components, Procedure, Simulation, Observations, Quiz and Certificate.

Main features:
- 2 separate subjects: EEE Virtual Lab and DELD Virtual Lab
- 24 experiments total
- Cards + full experiment list/table
- MIT ADT logo and consistent branding
- Dashboard with progress, score and certificates
- Components with professional icons and values
- Interactive simulations with observation table saving
- Experiment-specific engineering-level quizzes
- Printable certificate after quiz submission

Note:
This version uses browser localStorage as prototype storage. Firebase can be connected later for online login and cloud database.
